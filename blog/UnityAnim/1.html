Importing Skeletal Animations, assuming it's all built
1.
a. Select imported file and in Inspector tab: Choose Rig
b. Choose Animation Type drop-down: Choose Humanoid for bipedal and Generic for all else
2.
a. Select Animations button/tab, if Unapplied imported settings window appears, click Apply
b. Check Import Animation
c. Click Apply in lower right corner of Inspector just above the Animation Preview

The imported animation clips should be child assets of imported model
