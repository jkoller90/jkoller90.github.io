Canvas Elements in Unity

Right click - UI - Canvas

Canvas is always overlayed over the camera no matter where it is in the scene (default render mode)

  • Screen Space - camera: requires one to link the render camera
  • camera size - canvas is part of the world space
  • world space - canvas is another object in the world (i.e. VR in which canvas should be in the scene)

2d view mode makes it easier to view and edit


Adding an image:

  1. Add an image to the canvas
  2. Import an image
  3. Click on image and change Texture Type to Sprite (2d and GUI)
  4. Resize to fit proportions while holding shift

Background image and scalability:

  1. New image: Add, import, set to sprite
  2. Set rect transform left, top, right, and bottom to 0
  3. Make gridding equal stretch and stretch on both axes
  4. Physically drag the anchor points to the corners
  5. Under image, press Set Native Size
  6. May like to preserve aspect ratio check box as well
